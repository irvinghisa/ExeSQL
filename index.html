About this library
This library was created to facilitate the execution of SQL statements for different types of databases which are still working on this project to enhance its functionality.
Why this library
Why use this library? They have to ask the answer is simple, the library allows the execution of judgments on any computer regardless of the client version that you have installed, so this also helps teams work because it is not necessary for everyone the same version of the client to execute the statements.
