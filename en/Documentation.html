ExecProcedures (Class)

Exposes methods for different executions of stored procedures to query, insert or update.

Syntax

public class ExecProcedures

Constructors

Name Description
ExecProcedures(string) Initializes a new instance of the class ExecProcedures.
ExecProcedures(string, string) Initializes a new instance of the class ExecProcedures with the library directory to be referenced as Xcopy version of Oracle.

Properties

Name Description
Tables Variable that contains the tables of the inquiry. Added since the version 1.0.1.5
Values Variable that contains the array of parameter names with its output value. Added since the version 1.0.1.5
ds Variable that contains the records of the inquiry. Removed since the version 1.0.1.5
err Variable that contains the error message generated by executing a stored procedure.
exception Variable that contains the exception generated by executing a stored procedure.

Methods

Name Description
Get(string) Executes the stored procedure execute a query, no need input parameters.
Get(string[], string[], string) Executes the stored procedure execute a query with the input parameters sent.
Get(List<string>, List<string>, string) Executes the stored procedure execute a query with the input parameters sent. Added since the version 1.0.1.5
GetM(string) Executes the stored procedure that executes multiple queries, no need input parameters.
GetM(string[], string[], string) Executes the stored procedure that executes multiple queries with the input parameters sent.
GetM(List<string>, List<string>, string) Executes the stored procedure that executes multiple queries with the input parameters sent. Added since the version 1.0.1.5
GetSet(string[], string[], string) Executes the stored procedure that executes an insert, update and execute a query with the input parameters sent.
GetSet(List<string>, List<string>, string) Executes the stored procedure that executes an insert, update and execute a query with the input parameters sent. Added since the version 1.0.1.5
GetSetM(string[], string[], string) Executes the stored procedure that executes an insert, update and execute multiple queries with the input parameters sent.
GetSetM(List<string>, List<string>, string) Executes the stored procedure that executes an insert, update and execute multiple queries with the input parameters sent. Added since the version 1.0.1.5
GetValue(string) Executes the stored procedure which returns the value indicated in the procedure that is stored in the variable values, no need input parameters. Added since the version 1.0.1.5
GetValue(string[], string[], string) Executes the stored procedure which returns the value indicated in the procedure that is stored in the variable values, with only input or input-output parameters. Added since the version 1.0.1.5
GetValue(List<string>, List<string>, string) Executes the stored procedure which returns the value indicated in the procedure that is stored in the variable values, with only input or input-output parameters. Added since the version 1.0.1.5
GetValues(string) Executes the stored procedure which returns the values indicated in the procedure that is stored in the variable values, no need input parameters. Added since the version 1.0.1.5
GetValues(string[], string[], string) Executes the stored procedure which returns the values indicated in the procedure that is stored in the variable values, with only input or input-output parameters. Added since the version 1.0.1.5
GetValues(List<string>, List<string>, string) Executes the stored procedure which returns the values indicated in the procedure that is stored in the variable values, with only input or input-output parameters. Added since the version 1.0.1.5
Set(string) Executes the stored procedure that executes an insert or an update or both.
Set(string[], string[], string) Executes the stored procedure that executes an insert or an update or both, with the input parameters sent.
Set(List<string>, List<string>, string) Executes the stored procedure that executes an insert or an update or both, with the input parameters sent. Added since the version 1.0.1.5

Examples

  • C#
  • VB
  • Oracle SP 
using ExeSQL;

class ConsoleApplication
{
    static void Main()
    {
        string strcon = "CONNECTION_STRING";
        string proc = "NAME_STORE_PROCEDURE" or "NAME_PACKAGE.NAME_PROCEDURE";
        ExecProcedures ep = new ExecProcedures(strcon);
        if(ep.Get(proc))
        {
            //Code we use when execution of the procedure is successful
            //Before v1.0.1.5
            var dt = ep.ds.Tables[0];
            //Since v1.0.1.5
            var dt = ep.Tables[0];
        }
        else
        {
            //Code used when implementing the procedure fails
            string error = ep.err;
            Exception ex = ep.exception;
        }
    }
}

Imports ExeSQL

    Sub Main()
